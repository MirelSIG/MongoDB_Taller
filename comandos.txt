1. Podemos usar mongoshell para interactuar con MongoDB a través de la terminal.
2. Tambien podemos instalar MongoDB Compass (Desktop), una interfaz gráfica para administrar MongoDB. Puedes descargarlo 
desde el siguiente enlace:  https://www.mongodb.com/try/download/compass
Una vez descargado e instalado, puedes abrir MongoDB Compass y conectarte a tu instancia de MongoDB. 
Para conectarte, ingresa la URI de conexión (por ejemplo, mongodb://localhost:27017) y haz clic en "Connect". 
Esto te permitirá administrar tus bases de datos y colecciones a través de una interfaz gráfica.

Comandos básicos de MongoDB
1. Ejecuta: mongosh
2. Para mostrar las bases de datos disponibles:
    show dbs
3. Para crear o abrir BBDD deseada:
    use penascal-matricula
4. Para conocer el nombre de la BBDD en la que nos encontramos:
    db.getName()
    CONSULTAS POR SEDES
5. Para contar la cantidad de documentos en la coleccion una vez dentro de la DDBB:
    db["penascal-bilbao"].countDocuments()
6. Para listar los primeros 5 documentos de la co;leccion:
    db["penascal-bilbao"].find().limit(5).
Esto devuelve 5 objetos en el orden que tenga mongodb en su sistema.
    db["penascal-bilbao"].find().sort({ _id: 1 }).limit(5)
7. Para que entregue en orden secuencial completo y ascendente por _id ejecutamos:
    db["penascal-bilbao"].find().sort({ _id: 1 })
8. Para que entregue en orden secuencial completo y descendente por _id ejecutamos:
    db["penascal-bilbao"].find().sort({ _id: -1 })
9. Para que entregue en orden secuencial por fecha
    db["penascal-bilbao"].find().sort({ fecha: 1 })
10. Para que entregue en orden secuencial por nombre
    db["penascal-bilbao"].find().sort({ nombre: 1 })

CONSULTAS CRUZADAS ENTRE SEDES BILBAO Y VITORIA.  

11. Contar matriculas activas por SEDES
db["penascal-bilbao"].aggregate([
  { $match: { estado: "Activa" } },
  { $count: "Activas_Bilbao" }
])

db["penascal-vitoria"].aggregate([
  { $match: { estado: "Activa" } },
  { $count: "Activas_Vitoria" }
])
12. Uniendo ambas SEDES
db["penascal-bilbao"].aggregate([
  { $unionWith: "penascal-vitoria" }
])

18. Hacer un join ($lookup):
db["penascal-bilbao"].aggregate([
  {
    $lookup: {
      from: "penascal-docentes.profesores",
      localField: "profesor_id",
      foreignField: "id",
      as: "docente"
    }
  }
])
UNIR DATOS ENTRE DISTINTAS DDBB
19. Agregamos el campo docente a cada estudiante de penascal-bilbao:
db["penascal-bilbao"].aggregate([
  {
    $lookup: {
      from: "penascal-docentes.profesores",
      localField: "profesor_id",
      foreignField: "id",
      as: "docente"
    }
  }
])
20. Matriculas + docentes
db["penascal-bilbao"].aggregate([
  {
    $lookup: {
      from: "penascal-docentes.profesores",
      localField: "curso",
      foreignField: "departamento",
      as: "docente_asignado"
    }
  }
])

21. proveedores + socios por ciudad
db.getSiblingDB("penascal-proveedores").proveedores.aggregate([
{ $lookup: { from: "socios", localField: "ciudad", foreignField: "ciudad", as: "socios_relacionados" } }
